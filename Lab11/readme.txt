Describe el archivo package.json.
<br><br>

Al principio solamente se define el nombre y la info general del programa
como el nombre, la version, descripcion etc..

Despues se ponen los scripts que se desean agregar para que se pueda acceder
a ellos en un futuro junto con el repositorio, palabras clave
para buscar el programa.
Ahi mismo se integran el autor para la licencia del codigo,
el repositorio y las librerias que se hayan instalado.
