Validador de Passwords



¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?
| Porque si no se hace, el servidor puede recibir datos que no son válidos y esto puede causar problemas.

¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?
Puedes saltarte la seguridad de las validaciones de JavaScript si usas un navegador que no soporte JavaScript o si deshabilitas JavaScript en tu navegador.

Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?
Porque aunque se pueda saltar la seguridad de las validaciones de JavaScript, es una buena práctica usarlas porque si el usuario no tiene JavaScript habilitado, el servidor puede recibir datos que no son válidos y esto puede causar problemas.
